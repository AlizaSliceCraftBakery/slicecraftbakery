How to create a Personal Access Token

A token lets the admin panel upload images to your GitHub repo on your behalf. It only needs permission to write files.

1
Go to github.com/settings/tokens/new (you must be logged in)
2
Give it a name like Slice Craft Admin
3
Set expiration to No expiration (or 1 year)
4
Under Scopes, tick only repo (the top checkbox — this covers file writes)
5
Click Generate token — copy it immediately, GitHub only shows it once
6
Paste it into the token field above and click Connect & Test
